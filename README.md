# Student Management System - main version

## 📌 Project Description
This is a simple Student Management System built using HTML, CSS, and JavaScript.  
It allows basic structure for managing student data and demonstrates Git workflow.

---

## 🧠 Git Commands Used & Explanation

### 1. git init
Initializes a new Git repository.

### 2. git add .
Adds all files to staging area.

### 3. git commit -m "message"
Saves changes to Git history.

### 4. git branch
Creates a new branch for feature development.

### 5. git checkout
Switches between branches.

### 6. git merge
Combines two branches together.

### 7. git log
Shows commit history.

### 8. git diff
Shows changes between files.
